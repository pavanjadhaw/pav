### Usage

My npm namecard

```console
npx pav
```

## Setup

```sh
git clone https://github.com/pavanjadhaw/pav
cd pav && npm i
```

## Run

```sh
npm start
```

## Test

```sh
npm test
```

## Build

```sh
npm run prepare
```

## Dev

```sh
npm run dev
```
